## Building & running

    git clone https://github.com/ggerganov/wave-gui
    cd wave-gui
    git submodule update --init
    make
    ./build/main/wave-gui

## Dependencies

- [GLFW3](http://www.glfw.org)
- [FFTW](http://www.fftw.org)
- [SDL2](https://www.libsdl.org)
- [ImGui](https://github.com/ocornut/imgui) (git submodule)
